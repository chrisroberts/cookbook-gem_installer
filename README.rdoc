== Gem Installer

Simple cookbook for managing gem versions on a node.

=== Usage

Define gems (and optionally version restrictions) within the role:

  override_attributes(
    :gem_installer => [
      'pg',
      ['activerecord', '~> 3.0.0'],
      'ssmtp'
    ]
  )
